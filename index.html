Explorando Conceitos de Design e CSS

1. Affordance

Affordance é o conceito de como os objetos ou interfaces sugerem suas funções ao usuário. Em design, isso se refere a características de um objeto que indicam como ele deve ser usado. Por exemplo, um botão com um design que sugira que pode ser pressionado.

2. Semiótica

Semiótica é o estudo dos signos e símbolos e como eles são usados para comunicar significados. No design, isso se aplica ao uso de símbolos, ícones e imagens para transmitir informações de maneira visual.

3. Usabilidade

Usabilidade é a facilidade com que um usuário pode interagir com uma interface. Ela abrange aspectos como eficiência, aprendizado e satisfação do usuário durante o uso de um produto ou serviço.

4. CSS (Cascading Style Sheets)

CSS é uma linguagem usada para descrever a apresentação de um documento escrito em HTML. Ela define como os elementos devem ser exibidos na tela, no papel ou em outras mídias.

Formas de Incluir CSS em um Documento HTML

Existem três formas principais de incluir CSS em um documento HTML:

  • CSS Inline: Usado diretamente no elemento HTML com o atributo style. Exemplo: <div style="color: red;">Texto</div>. Ideal para estilos rápidos e simples.
  • CSS Interno: Colocado dentro da tag <style> na seção <head>. Exemplo: 
    <style>
    p { color: blue; }
</style>
    Ideal para estilos específicos de uma página.
  • CSS Externo: Referenciado através de um arquivo externo com a tag <link>. Exemplo: 
    <link rel="stylesheet" href="style.css">
    Ideal para manter o código limpo e reutilizável.
